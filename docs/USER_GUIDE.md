# User Guide

## Logging In

1.  Open the application in your browser
1.  Click "Login with SailPoint"
1.  Authenticate with your SailPoint credentials
1.  You will be redirected back to the application

## Navigation

Use the top navigation bar to switch between components:

-   **Home** — Dashboard with quick stats
-   **Identities** — Browse and search identities
-   **Accounts** — Browse and filter accounts
-   **Search** — Global search across all indexes

## Component Settings

Click the settings icon to enable or disable components. Changes take effect immediately and persist across sessions.
